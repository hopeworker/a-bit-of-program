# 英文聊聊吧

英文播客: "英文聊聊吧"

**开通啦！**

通过英文交流的形式，分享我们的生活故事，同时提高英文口语和听力水平。

每个聊天话题分两集录制。  
第一集就一个话题用英文轻松聊天，分享我们的生活故事。  
第二集通过对第一集聊天过程的英文部分进行反思，纠正，重新录制一遍。
这里是我们的英文聊聊“吧”。

可以通过以下几种方式收听播客。
- [喜马拉雅](https://www.ximalaya.com/album/86658945.xml)

- 微信公众号: 在“小熊的公众号”里，点击 “英文播客” 菜单。
