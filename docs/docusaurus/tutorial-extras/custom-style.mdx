# 样式配置

## 通过custom.css
在 docusaurus.config.ts 中配置了样式设置。

```jsx title="docusaurus.config.ts"
        theme: {
          customCss: './src/css/custom.css',
        },
```

## Docusaurus use infima css framework

- https://infima.dev/

For example, if you want to add Tabs for your page.
- https://docusaurus.io/docs/markdown-features/tabs

You can use “tabs tabs--block" to style tabs.  
use ”button button--primary button--lg“ to style button.

````jsx
<Tabs className="tabs tabs--block">
  <TabItem value="tab1" label="Tab 1" default>
    <h3>Content for Tab 1</h3>
      <Link className="button button--primary button--lg" to="/docs/main">
        点滴积累
      </Link>
  </TabItem>
  <TabItem value="tab2" label="Tab 2">
    <h3>Content for Tab 2</h3>
  </TabItem>
</Tabs>
````
